# GLARIX Eyewear

A responsive multi-page e-commerce website built using HTML, CSS, and Vanilla JavaScript.

## Features

- Home page with hero section
- Product listing with filter and sorting
- Product detail page
- Related products carousel
- Cart system with item count badge
- Order confirmation page
- Fully responsive design
- Uses localStorage for cart persistence

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

Download or clone the repository and open `index.html` in any modern web browser.

No backend or installation required.
